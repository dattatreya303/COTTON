Nqueens was ported from the B\+O\+TS nqueens.\+c benchmark. See below for provenance. This program computes all solutions to the n-\/queens problem where n is specified in argv\mbox{[}1\mbox{]} (default = 11) The program uses the count of the total number of solutions as a correctness check and also prints the execution time for each repetition. 

Note the use of single \char`\"{}finish\char`\"{} statement in \mbox{\hyperlink{nqueens_8cpp_a0ddf1224851353fc92bfbff6f499fa97}{main()}} that awaits termination of all async\textquotesingle{}s created by the recursive calls to nqueens\+\_\+kernel. 

To study scalability on a multi-\/core processor, you can execute \char`\"{}\+C\+O\+T\+T\+O\+N\+\_\+\+W\+O\+R\+K\+E\+R\+S=$<$\+N$>$ ./nqueens 11\char`\"{} by varying the number of worker (=N) threads.
